A cheeky little article about CSS. Mostly as a reminder to myself why I've always found CSS so interesting and fun to work with. CSS was such a huge part of me falling in love with web design and development over 20 years ago.

Granted, it was heck of a buggy to use in the early 2000's. Especially cause browser vendors had their quirks - which meant you had to figure out how to make it work in all the common browsers. But that was part of the fun. It was like a puzzle to solve.

Remember this?...

```css
.clearfix::after {
  content: "";
  display: table;
  clear: both;
}
```

This was such a familiar piece of CSS that I used all the time. It was a little 'hack' to clear floats in a container.

And then came along `flexbox`.
